# Dominio-automatizado-win
Esse script automatiza três etapas essenciais no gerenciamento de computadores via Active Directory: ele ingressa a máquina no domínio, move o objeto para a Unidade Organizacional (OU) definida e adiciona o computador ao grupo especificado, garantindo que a estrutura hierárquica e as políticas de segurança sejam aplicadas corretamente.
